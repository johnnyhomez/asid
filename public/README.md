# Public Assets

This directory contains static assets for the ASID website.

## Required Assets (to be added):

- **favicon.ico** - Website icon (16x16 or 32x32 px)
- **og-image.jpg** - Open Graph image for social media sharing (1200x630 px)
- **apple-touch-icon.png** - iOS home screen icon (180x180 px)

## Recommended Content:

- Images of the facility
- Photos of activities (with proper permissions)
- Branding assets and logos

## Note:
All images should be optimized for web use to ensure fast page load times.
